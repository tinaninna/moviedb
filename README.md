# Netflix Clone 🎬

A responsive Netflix Clone built using modern web technologies. This project replicates the Netflix UI and displays movies/TV shows using an API.

---

## 🚀 Features

- Netflix-style homepage UI
- Movie & TV show listings
- Trending and categorized sections
- Responsive design (mobile + desktop)
- Dynamic data fetching (API-based)
- Smooth navigation experience

---

## 🛠️ Tech Stack

- React.js / Next.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Tailwind CSS / Bootstrap (if used)
- Axios / Fetch API
- TMDB API (if used)
